## CVE-2025-21298

[CVE-2025-21298](https://www.offsec.com/blog/cve-2025-21298/) is critical Remote Code Execution(RCE) vulnerability in `Microsoft Windows OLE`.An attacker can exploit this vulnerability through a specially crafted email or `RTF` document containing a malicious OLR object.Successful exploitation may allow arbitrary code execution on the victim's system.During the investigation, indicators associated with `CVE-2025-21298` exploitation were identified and analyzed.The malicious activity was confirmed and appropriate containment were taken.


I found `IoC` in VirusTotal

### Process Chain Breakdown

- `WINWORD.EXE /q` silently launches the malicious RTF/DOC file; the `/q` flag suppresses Word's main GUI window, typical for exploit execution.
- The exploit corrupts Word memory to force an application crash, spawning legitimate Office crash handler `DW20.EXE - s 3244` (Dr.Watson) and Windows native crash utility `dwwin.exe`.Attackers abuse the vulnerable crash handler's process privileges to escape Office sandbox.
- Abused `dwwin.exe` spawns `cmd.exe + conhost`, enabling attacker shell access to run post-exploitation commands.
- Malware side-loads payload into trusted system processes (`audiodg.exe`, `taskhost.exe`) for stealthy persistence and evasion.A second separate `WINWORD.EXE (Office14/2010)` instansce spawns as secondary exploit staging


## Post-Exploit Payload Execution
